Admin Actions

Please select an option from the navigation menu to perform an action:

  • Add Carrier: Register a new airline carrier to the system.
  • Add Flight: Add a new flight to the system.
  • Edit Carrier: Modify the details of an existing carrier.
  • Remove Carrier: Remove a carrier from the system.
  • Flight Cancellation - Refund Calculation: Handle flight cancellations and calculate refund amounts.
  • View Carriers: View the list of all registered airline carriers.
  • View Flights: View the list of all scheduled flights.
  • Edit Flights: Modify details of existing flights.
